# Premium Modern Login & Register UI

Halaman Login dan Register yang modern, responsif, dan elegan dengan desain **Glassmorphism**. Proyek ini dibuat menggunakan HTML kustom dan CSS murni untuk memberikan tampilan yang premium.

## ✨ Fitur Utama

- **Desain Glassmorphism**: Tampilan transparan yang modern dengan efek blur di latar belakang.
- **Fully Responsive**: Tampilan tetap rapi mulai dari desktop hingga layar mobile (smartphone).
- **Animasi Halus**: Efek transisi _fade-in_ dan _slide-up_ saat halaman dimuat.
- **Halaman Lengkap**: Sudah termasuk halaman Login (`index.html`) dan Register (`register.html`).

## 🛠️ Teknologi yang Digunakan

- **HTML5**: Struktur halaman yang semantik.
- **CSS3**: Desain modern menggunakan CSS Variables, Flexbox, dan Keyframes.
- **Google Fonts**: Menggunakan font 'Outfit' untuk tampilan yang profesional.

## 🚀 Cara Menjalankan

1. Clone repositori ini atau download filenya.
2. Buka file `index.html` di browser favorit Anda.
3. Klik tombol "Sign up" untuk melihat halaman registrasi.
